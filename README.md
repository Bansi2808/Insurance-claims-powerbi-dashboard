## 🚀 Overview

This project presents an interactive **Insurance Claims Performance Dashboard** built using Power BI.

It analyzes claim trends, fraud detection, incident patterns, and financial impact to support **data-driven decision-making**.

---

## 🎯 Key Features

- 📊 Total Claims & Claim Amount KPIs  
- ⚠️ Fraud Detection Analysis  
- 📈 State-wise Claims Distribution  
- 📉 Claim Amount Approval Trends  
- 🕒 Incident Analysis by Hour  
- 🚗 Vehicle Age vs Incident Analysis  
- 👥 Gender-based Insights  
- 📆 Time-based Trends (Year, Quarter, Month)

- 📈 Peak incident hour identified around **54 incidents**
- 💰 Total claim amount exceeds **53M**
